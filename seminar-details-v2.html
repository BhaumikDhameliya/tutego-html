Object orientation

Java

Because they are platform independent, Java programs can run on different computer systems. Its high performance and robustness make Java a popular language for critical server components. In this seminar, participants will get to know the object-oriented language Java using the current version. The training focuses on both syntax and object-oriented design with Java and illustrates concepts such as polymorphism and automatic garbage collection in many practical examples. After the Java training, participants have a good overview of the Java packages with their classes, interfaces and enumerations and can program platform-independent applications.

The course provides optimal preparation for the advanced Java seminar, which deepens the Java standard libraries and provides an overview of Java EE.
